# kWoodcutterAn OSMB script for powerchopping trees in popular locations. Select your tree type via a clean GUI, chops continuously, and automatically drops logs and clue/scroll boxes for maximum XP/hr with zero babysitting.### Requirements* Any axe (equipped or in inventory)* Start near your selected tree location* Visible chat messages (for "inventory too full" detection)### Tree Locations (Selected via GUI)The script supports 5 high-efficiency spots:* Normal trees (Varrock West)* Oaks (Varrock West)* Willows (Draynor Village)* Willows (Barbarian Assault)* Maples (Seers Village)### Features* **Smart Tree Detection**:  * Pixel-based stump filtering (purple square on stumps)  * Closest tree targeting with convex hull interaction  * 20s stump timeout for fast recovery* **Intelligent Dropping**:  * Fixed threshold at 26 logs  * Automatically drops all clue/scroll boxes (all tiers: beginner to master)  * Chat detection backup for "Your inventory is too full to hold any more logs"* **Quality of Life**:  * Region safety check – stops if in wrong location  * Startup full inventory handling – drops immediately if needed  * Sleek compact paint with live stats* **Paint Overlay**:  * Title: "KWoodcutter"  * Status (Chopping/Dropping)  * Current Tree Type  * Runtime  * Accurate XP/hr (based on dropped logs)### Setup1. Travel to your desired tree location2. Start the script3. In the GUI, select your tree type from the dropdown4. Click "START SCRIPT"5. The script will begin chopping and handle dropping automatically### Paint DisplayThe compact modern overlay shows:* Title: "KWoodcutter"* Status: Chopping / Dropping* Tree: Current selected tree* Runtime: HH:MM:SS* XP/hr: Real-time accurate rate### Notes* Optimized for pure powerchopping – drops logs and clue/scroll boxes, nests/seeds accumulate harmlessly* Best XP/hr on Maples (Seers Village) at higher levels* Script stops automatically if you leave the selected region* For longest runs, use high FPS and ensure chat messages are visible### Version1.0
